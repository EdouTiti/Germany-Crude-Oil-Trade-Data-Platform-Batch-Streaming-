# Project Title: Germany Crude Oil Trade Data Platform (Batch)

## Project Overview
This project builds an end-to-end data pipeline to analyze Germany’s crude oil trade from 2019 to 2024.
The goal is to combine batch processing and real-time streaming to produce analytics-ready data and interactive dashboards.

## Problem Statement
Crude oil trade data is often available as raw files but not structured for analytics or real-time monitoring.  
This project transforms historical and simulated real-time data into a structured format to support:  
- Trend analysis
- Country-level insights
- Real-time monitoring
